# 🌍 Climate Trend Analyzer

## 📌 Project Overview

Climate Trend Analyzer is a data science project that analyzes historical climate data to identify patterns, trends, and anomalies over time.

This project simulates real-world climate data and applies data analysis techniques to understand:

* Temperature trends 🌡️
* CO₂ level rise 🌫️
* Rainfall patterns 🌧️
* Unusual climate events (anomalies) 🚨

---

## 🚀 Features

* 📊 Data Generation (synthetic climate dataset)
* 🧹 Data Cleaning & Preprocessing
* 📈 Data Visualization (graphs & heatmaps)
* 🚨 Anomaly Detection using statistical methods
* 📁 Organized project structure

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn

## ▶️ How to Run the Project

### Step 1: Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Climate-Trend-Analyzer.git
cd Climate-Trend-Analyzer
```

### Step 2: Install dependencies

```
pip install pandas numpy matplotlib seaborn
```

### Step 3: Run scripts

```
python src/generate_data.py
python src/preprocess.py
python src/visualize.py
python src/anomaly_detection.py
```
